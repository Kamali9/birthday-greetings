Vite - v 7
React - v 18
tailwindcss - 3.4
react-icons - 5.5
react-confetti - 6.4
postcss - 8.5

API server: npm install -g json-server
tailwind css: npm install -D tailwindcss@3 postcss autoprefixer
npx tailwindcss init -p

Local start: npm run dev
api start: json-server --watch public/birthday.json --port 3006

Install node packages: npm install
